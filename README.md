# XLSuiteVolley — Download e installazione

Software di scout e analisi pallavolo (Windows 64-bit).

## ⬇️ Scarica l'ultima versione

👉 **[Vai all'ultima release](../../releases/latest)** e scarica il file
`XLSuiteVolleySetup-x.y.z.exe`, poi eseguilo.

## Requisiti
- Windows 10/11 a **64 bit**.
- L'installer è autoconsistente: se mancano, installa automaticamente i
  prerequisiti (Access Database Engine x64, Visual C++ Redistributable x64,
  LibreOffice x64). Non serve avere Microsoft Excel.

## Installazione
1. Scarica l'installer dalla pagina delle release.
2. Doppio clic ed esegui (Windows potrebbe chiedere conferma SmartScreen → *Ulteriori informazioni* → *Esegui comunque*).
3. Segui la procedura guidata.

## Attivazione / licenza
Il software richiede attivazione. Per ottenere la licenza contatta l'autore
fornendo il codice mostrato dall'applicazione al primo avvio.

## Aggiornamenti (FIX)
Le correzioni vengono pubblicate qui come nuove release. Per aggiornare,
scarica ed esegui l'ultimo installer di tipo *Update*.

## ❓ Domande frequenti e risoluzione problemi

### Windows / l'antivirus blocca l'installer
È un falso allarme: l'app non ha una firma digitale a pagamento, ma è sicura.
- Alla schermata azzurra *"Windows ha protetto il PC"* → **Ulteriori informazioni → Esegui comunque**.
- Se il browser segnala il download come rischioso → **Mantieni / Conserva comunque**.

### All'avvio compare un errore su un componente mancante
Riesegui l'installer per completare l'installazione dei prerequisiti
(Access Database Engine x64, Visual C++ x64, LibreOffice x64). Riavvia il PC se richiesto.

### Le stampe o i PDF non vengono generati
Non serve Microsoft Excel. Per produrre i PDF il programma usa **LibreOffice**, che
l'installer installa automaticamente: assicurati di aver completato l'installazione.

### I video / i montaggi non funzionano
Gli strumenti video (ffmpeg/ffprobe) sono inclusi nell'installer: non va installato nulla a parte.

### Come aggiorno a una nuova versione (FIX)
Scarica ed esegui l'ultimo installer da questa pagina
([releases/latest](../../releases/latest)). I tuoi dati e scout non vengono toccati.

### Il download è molto lento
Il file è grande (~900 MB): usa una connessione stabile. Dal browser puoi mettere
in pausa e riprendere il download.

## 💻 Requisiti di sistema
- Windows 10 o 11 a **64 bit**
- ~2 GB di spazio libero (più lo spazio per scout e video)
- Permessi di amministratore per l'installazione

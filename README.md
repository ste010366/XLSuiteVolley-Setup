# XLSuiteVolley — Download e installazione

Software di scout e analisi pallavolo (Windows 64-bit).

## ⬇️ Scarica l'ultima versione

👉 **[Vai all'ultima release](../../releases/latest)** e scarica il file
`XLSuiteVolleySetup-x.y.z.exe`, poi eseguilo.

## Requisiti
- Windows 10/11 a **64 bit**.
- L'installer è autoconsistente: se mancano, installa automaticamente i
  prerequisiti (Access Database Engine x64, Visual C++ Redistributable x64,
  LibreOffice x64). Non serve avere Microsoft Excel.

## Installazione
1. Scarica l'installer dalla pagina delle release.
2. Doppio clic ed esegui (Windows potrebbe chiedere conferma SmartScreen → *Ulteriori informazioni* → *Esegui comunque*).
3. Segui la procedura guidata.

## Attivazione / licenza
Il software richiede attivazione. Per ottenere la licenza contatta l'autore
fornendo il codice mostrato dall'applicazione al primo avvio.

## Aggiornamenti (FIX)
Le correzioni vengono pubblicate qui come nuove release. Per aggiornare,
scarica ed esegui l'ultimo installer di tipo *Update*.

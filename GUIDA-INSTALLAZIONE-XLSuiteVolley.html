XLSuiteVolley — Guida all'installazione

Questa guida ti accompagna passo passo nello scaricare e installare XLSuiteVolley sul tuo computer. Bastano pochi minuti.

Cosa serve:
  • Un PC con Windows 10 o 11 a 64 bit.
  • Una connessione a Internet per il download (il file pesa circa 900 MB).
  • I permessi di amministratore del PC (di solito sul proprio computer ci sono già).
Non serve avere Microsoft Excel: il programma funziona da solo.

Passo 1 — Scarica il programma

1 Apri questo indirizzo nel browser:

https://github.com/ste010366/XLSuiteVolley-Setup/releases/latest

2 In fondo alla pagina, nella sezione Assets, clicca sul file XLSuiteVolleySetup-1.0.1.exe per scaricarlo.

3 Attendi la fine del download (qualche minuto, secondo la tua connessione). Di solito il file finisce nella cartella Download.

Passo 2 — Avvia l'installazione

1 Fai doppio clic sul file scaricato XLSuiteVolleySetup-1.0.1.exe.

⚠ Importante — avviso di Windows
Può comparire una schermata azzurra: “Windows ha protetto il PC”. È normale (il programma non ha la firma digitale a pagamento, ma è sicuro).
Fai così: clicca su “Ulteriori informazioni” e poi sul pulsante “Esegui comunque”.

2 Quando Windows chiede i permessi di amministratore, clicca Sì.

3 Segui la procedura guidata cliccando Avanti fino a Installa.

Passo 3 — Attendi il completamento

La prima installazione può richiedere alcuni minuti: il programma prepara da solo i componenti aggiuntivi necessari. In alcuni casi può chiedere di riavviare il PC: conferma pure e, dopo il riavvio, l'installazione si completa.

✓ Fatto! Al termine troverai l'icona di XLSuiteVolley sul Desktop e nel menu Start.

Passo 4 — Primo avvio e attivazione

1 Avvia XLSuiteVolley con un doppio clic sull'icona.

2 Al primo avvio l'applicazione mostrerà un codice identificativo del tuo computer.

3 Inviami quel codice (vedi i contatti in fondo): ti rimanderò il file di licenza con le istruzioni per inserirlo e sbloccare il programma.

Problemi comuni

SintomoSoluzione
L'antivirus o Windows blocca il file È un falso allarme dovuto alla mancanza di firma digitale. Scegli “Esegui comunque” / “Mantieni” / “Consenti”.
All'avvio compare un errore su un componente mancante Riesegui l'installer: completa l'installazione dei componenti aggiuntivi. Riavvia il PC se richiesto.
Le stampe / i PDF non si generano Non serve Excel. Per i PDF il programma usa LibreOffice, installato automaticamente: se l'installazione è stata completata, funziona.
Il download è molto lento Il file è grande (~900 MB). Usa una connessione stabile e attendi; puoi mettere in pausa e riprendere dal browser.
